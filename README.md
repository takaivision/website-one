# Takai Vision Technologies Pvt Ltd — Website

This is a simple static website (HTML, CSS, JS) for Takai Vision Technologies Pvt Ltd.

How to preview

1. Open `index.html` in your browser, or serve the folder with a simple HTTP server:

```sh
# from the website folder
python3 -m http.server 8000
# then open http://localhost:8000
```

Files

- `index.html` — Main page
- `css/styles.css` — Styles
- `js/main.js` — Small JS interactions (nav, form)
